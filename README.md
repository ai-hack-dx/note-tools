# note-tools

note記事用に作成した、ブラウザだけで完結する無料Webツール集です。インストール不要で、リンクを開くだけで使えます。

公開ページ: https://ai-hack-dx.github.io/note-tools/

## ツール一覧

| ツール | 説明 | リンク |
| --- | --- | --- |
| 取扱説明書メーカー | 自分・商品の「取扱説明書」を作成 | [torisetsu-maker.html](https://ai-hack-dx.github.io/note-tools/torisetsu-maker.html) |
| 相性診断メーカー | 2人の相性を診断するカードを生成 | [aisho-maker.html](https://ai-hack-dx.github.io/note-tools/aisho-maker.html) |
| ペット擬人化メーカー | うちの子を擬人化キャラに変換 | [pet-maker.html](https://ai-hack-dx.github.io/note-tools/pet-maker.html) |
| 画像コンバーター | 画像のリサイズ・形式変換 | [image-converter.html](https://ai-hack-dx.github.io/note-tools/image-converter.html) |
| PDFエディター | PDFの結合・分割などの編集 | [pdf-editor.html](https://ai-hack-dx.github.io/note-tools/pdf-editor.html) |
